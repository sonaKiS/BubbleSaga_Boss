Upgrade project materials shaders to URP or HDRP using packages inside the "HDRP and URP asset" folder.

Or you can do it manually for URP:
OUse standard upgrade tool to change standard materials to URP:
Edit > Rendering > Materials > Convert All Built-In Materials to URP
In Unity 2021.2 and above, navigate to:
Window > Rendering > Render Pipeline Converter
